# SeCommerce API

## Getting Started

### Dependencies

* Create environtment
```sh
$ python3 -m venv env
```
OR

```sh
$ pip install virtualenv
$ virtualenv -p python3 env
```

* Install dependencies

```sh
$ source env/bin/activate
```

### Installing 

* How to run the requirements

```sh
$ python -m pip install -r requirements/dev.txt
```
